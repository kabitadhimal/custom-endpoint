This plugin shows the jobs listing as seen in https://qkly.io/ and user can apply to the job.

== Description ==
This plugin shows the jobs listing as seen in https://qkly.io/. The partner company of https://qkly.io/ will need to provide their unique company id in the backend to show the jobs. After the company id and other options are configured in the backend, the plugin will also show the filters to search jobs.

The options in the backend are
-  number of jobs per page
- layout for the job list
- company id

The filter in the frontend will allow to search by job title, skills, keyword, company, job type and salary. And user can apply the job by clicking in apply link.

== Installation ==
1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click Qkly Job Listing Settings just after the settings to check the settings of the plugin
4. API URL of Qkly Jobs Plugin is https://jobs.qkly.io/
4. Other options on the settings are self explained

== How to use in the theme ==
To list the qkly use the shortcode in the editor as following

== shortcode to  list the qkly jobs  ==
[qkly_job_list]
